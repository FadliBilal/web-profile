Resume Pembelajaran HTML

Apa yang saya pelajari tentang dasar HTML dari website referensi yang saya baca.

HTML (HyperText Markup Language) adalah bahasa yang digunakan untuk membangun struktur halaman web. Dengan HTML, kita dapat menyusun elemen-elemen seperti teks, gambar, dan tautan sehingga dapat ditampilkan dengan benar di browser. Tag HTML seringkali terdiri dari elemen pembuka dan penutup, contohnya tag <p> untuk paragraf dan tag <a> untuk tautan.


Setiap dokumen HTML dimulai dengan deklarasi DOCTYPE html, diikuti oleh tag <html> yang membungkus seluruh konten halaman. Di dalam tag <html>, ada dua bagian utama : <head> dan <body>. Bagian <head> berisi informasi seperti judul halaman yang muncul di tab browser, sedangkan <body> menyimpan konten yang terlihat oleh pengguna, seperti teks dan gambar.


HTML juga memungkinkan integrasi dengan CSS (Cascading Style Sheets) untuk mengatur tampilan. Kombinasi HTML dan CSS memungkinkan pembuatan halaman web yang fungsional dan menarik. Untuk pemula, mempelajari dasar-dasar HTML adalah langkah awal yang penting sebelum beralih ke teknologi web yang lebih kompleks.
